1. Install NodeJs to your machine. Up-to-date latest/current version, not LTS.
  2. In the Terminal, run
    >[sudo] npm install -g npm
    Sudo is required for Mac/Linux.
  3. Install Angular CLI by running
    > [sudo] npm install -g @angular/cli

    4. Just FYI: What is and Why use Angular Command Line Interface?
    Angular CLI is a powerful tool set for creating, managing and building Angular Application.
    It also comes with simple command lines to build the project for production and so on.

    The reason to use CLI and not just import the angular framework into an existing index.html page is because Angular is a futuristic framework which uses features like typescript (a superset of javascript) which needs to be compiled before it runs in the browser. We have to take into account some of the extra steps that happens in the background during development and doing this manually can be a pain. In conclusion, Angular CLI is a great tool for big and very big projects! It allow developers to focus on Angular Code instead of workflow tasks!

    Just FYI: Reinstall to update CLI
    > [sudo] npm uninstall -g angular-cli @angular/cli
    > npm cache clean
    > [sudo] npm install -g @angular/cli
  4. Create new Angular Project by typing in "ng new project-name"
    > ng new my-app
    Similarity with rails news.
  5. Get into the project folder (cd dir-name) and Launch the application by running
    >ng serve
    and open http://localhost:4200/ on your browser. This is similar to rails server (rails) and http://localhost:3000/
